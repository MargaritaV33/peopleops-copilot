# PeopleOps Copilot

An agentic HR policy assistant built with Retrieval-Augmented Generation (RAG), hybrid retrieval, conversational memory, source citations, and tool calling.

## Author

Margarita Varla

## Project Goal

Build an end-to-end conversational assistant that helps employees:

- ask questions about company policies,
- retrieve grounded answers from a real policy knowledge base,
- view the sources used to generate each answer,
- maintain context across follow-up questions,
- retrieve policy documents,
- and create demo HR discussion requests through agent tools.

> This project is built as part of GitHub Project 2 for the AI Engineering Bootcamp.
